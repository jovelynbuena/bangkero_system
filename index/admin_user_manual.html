Sidebar Navigation Overview

All Roles

The sidebar on the left side of every admin page is the main navigation hub. It stays visible at all times on desktop and can be toggled on mobile.

Sidebar Controls

Hamburger / Collapse Button On desktop: collapses sidebar to icon-only mode. On mobile: opens or closes the sidebar drawer.
Dropdown Arrows Archive and Settings sections have a caret arrow. Click the parent item to expand or collapse the sub-menu.
User Pill (Top Right) Shows your name, avatar, and role. Informational only — no clickable actions.
Red Badge (Envelope) Appears on Contact Messages when there are unread messages. Number indicates the count.

Main

Dashboard

Admin · Officer · Member

The Dashboard is the first page you see after logging in. It shows a high-level summary of the entire association at a glance.

Summary Cards

  • Total Members – total registered members in the system
  • Active Officers – current officers with active terms
  • Upcoming Events – events scheduled in the future
  • Announcements – number of active announcements

Upcoming Events Widget

Lists the next upcoming events. Click an event name to go directly to its detail page.

Recent Announcements Widget

Shows the latest announcements. Click to read the full announcement.

The Dashboard is read-only. No data can be edited from here — use the dedicated management pages for edits.
Management

Members

Admin · Officer

Manage all registered association members — add, edit, view, export, and archive member records.

Main Buttons

Add Member Export Excel Print
  • Add Member – opens a form to register a new member (name, contact info, address, membership date, etc.)
  • Export Excel – downloads the current filtered member list as an Excel / CSV file
  • Print – opens a print-ready view of the member list

Per-Row Actions

View Edit Archive
  • View – shows full member profile including attendance history
  • Edit – update member information
  • Archive – moves member to the Archives section (not deleted, can be restored)

Search & Filter

Use the search box to find members by name or ID. Filter by status (Active / Inactive) or membership date range.

Officers

Admin · Officer

View the list of all officers — both current and past. Shows position, term dates, and contact information.

Buttons

View Profile Edit Archive
  • View Profile – see full officer details
  • Edit – update officer details (position, term dates)
  • Archive – archives the officer record (for past officers)
Officers are linked to Members. An officer record references a member's profile. Use Manage Officers to create new officer assignments.

Manage Officers

Admin Only

Assign members to officer positions and manage their system access roles (Treasurer, Secretary, etc.).

Buttons

Add Officer Edit Remove
  • Add Officer – select a member, assign a role/position, set term start and end dates, and set their Transparency access (Treasurer / Secretary / Both)
  • Edit – change role, term dates, or transparency access
  • Remove – removes the officer assignment (member stays in the system)
The Transparency Role field controls what an Officer can see in the Transparency page. Set to Treasurer for financial data, Secretary for programs, or Both for full access.

Roles

Admin Only

Create and manage officer role names (e.g., President, Vice President, Treasurer, Secretary).

Buttons

Add Role Edit Archive
  • Add Role – create a new named officer role
  • Edit – rename an existing role
  • Archive – hides the role from active use (still linked to past officers)

Awards

Admin Only

Record and manage awards given to members of the association.

Buttons

Add Award Edit Archive
  • Add Award – fill in award title, recipient (member), date, and description
  • Edit – update award details
  • Archive – moves award to Archives (restorable)
Activities

Events

Admin · Officer

Create, manage, and track association events. Members can be marked present/absent per event.

Main Buttons

Add Event Export PDF
  • Add Event – create a new event with title, date, time, venue, and description
  • Export PDF – generates a printable PDF of the event list

Per-Row Actions

Attendance Edit Archive
  • Attendance – opens the attendance sheet for that event; mark members as Present, Absent, or Excused
  • Edit – update event details
  • Archive – moves event to Archives

Attendance Reports

Admin · Officer

View aggregated attendance data across all events. Filter by member, date range, or event.

Buttons

Export PDF Export Excel Print
  • Export PDF – downloads the attendance report as a formatted PDF
  • Export Excel – downloads attendance data as a spreadsheet
  • Print – opens print dialog for the current report view

Filters

  • Filter by Event – select a specific event to see its attendance
  • Filter by Member – see all events a specific member attended
  • Filter by Date Range – narrow results to a period
  • Filter by Status – Present, Absent, or Excused

Galleries

Admin · Officer

Upload and manage photo galleries for events and activities. Photos are visible on the public-facing website gallery page.

Buttons

Add Gallery Export Selected
  • Add Gallery – create a new gallery album with a title, date, and upload multiple photos at once
  • Export Selected – download selected gallery images as a ZIP file

Per-Gallery Actions

View Edit Archive
  • View – opens the gallery to see all photos in a lightbox carousel
  • Edit – rename the gallery or add/remove photos
  • Archive – hides gallery from the public site

Announcements

Admin · Officer

Post and manage association announcements that appear on the member portal and public site.

Buttons

Add Announcement
  • Add Announcement – fill in title, body text, optional image, and publish date; toggle visibility (Members only / Public)

Per-Row Actions

Edit Archive
  • Edit – update announcement content or visibility
  • Archive – removes from public view (restorable from Archives)
Communication

Contact Messages

Admin · Officer

Read and manage messages sent by website visitors through the Contact Us form on the public site.

Badge Indicator

A red badge on the sidebar envelope icon shows the count of unread messages. It disappears once all messages are read.

Per-Row Actions

View / Mark Read Archive Delete
  • View / Mark Read – opens the message and marks it as read; the sidebar badge count decreases
  • Archive – moves message to archive (still accessible in Archives > Messages)
  • Delete – permanently removes the message

Filters

Filter by status (Unread / Read / All) and date range to manage your inbox efficiently.

Transparency

Transparency

Admin · Treasurer · Secretary

Record and publish financial assistance, expenses, and program achievements to promote accountability within the association.

Three Tabs

  • Assistance Tab – record donations and aid received (cash or in-kind); visible to Treasurer & Secretary
  • Expenses Tab – log association spending; visible to Treasurer
  • Achievements / Programs Tab – document completed programs with photos; visible to Secretary

Key Buttons

Add Assistance Add Expense Add Photo / Achievement Edit Archive
  • Each tab has its own Add button that opens a modal form
  • Edit (pencil icon) – update a record
  • Archive (box icon) – hides record from public transparency view
For a detailed guide with every modal field explained, see the separate Transparency User Manual at /index/management/transparency_user_manual.html
System

Backup & Restore

Admin Only

Protect the system by creating backups of the database or the entire project, and restore them when needed.

Four Main Buttons

Create Backup Now Restore Database Create System Backup Restore System
  • Create Backup Now – generates a .sql file with all database tables and data
  • Restore Database – upload a .sql file to overwrite the current database
  • Create System Backup – generates a .zip containing the database dump + all project files
  • Restore System – upload a .zip to restore both files and database

Backup History Actions (per row)

Download Restore Delete
  • Download – save the backup file to your computer
  • Restore – restore directly from the server-side file (no upload needed)
  • Delete – permanently delete the backup file from the server
Restore and Delete are irreversible. Always download a copy before deleting, and create a fresh backup before restoring.
For the full Backup & Restore guide, see /index/utilities/backup_user_manual.html

Activity Logs

Admin Only

View a complete audit trail of all important actions performed in the system — who did what and when.

What is Logged

  • Login and logout events
  • Member / Officer additions, edits, and deletions
  • Backup creation and restore operations
  • Transparency record changes
  • Announcement and event changes

Buttons

Clear Logs Export
  • Clear Logs – permanently deletes all log entries (use with caution)
  • Export – downloads logs as an Excel or CSV file for record-keeping

Filters

Filter logs by date range, action type, or user to find a specific event quickly.

Clearing logs is permanent. Export the logs before clearing if you need a record for auditing purposes.
Archives

Archives

Admin Only

The Archives dropdown contains a separate archive section for each module. Archived records are hidden from active use but are never deleted — they can always be restored.

Archive Sections

MembersArchived member profiles
OfficersPast / archived officers
EventsPast or cancelled events
AwardsArchived award records
GalleriesHidden gallery albums
MessagesArchived contact messages
RolesArchived officer role names
AccountsDeactivated user accounts
AnnouncementsArchived announcements
Website ContentPrevious website content versions
TransparencyArchived transparency records

Actions Available in Every Archive

Restore Delete Permanently
  • Restore – moves the record back to the active list
  • Delete Permanently – completely and irreversibly removes the record from the database
Permanent Delete cannot be undone. If unsure, leave the record archived instead of deleting it.
Settings

Profile Settings

Admin · Officer · Member

Update your personal account information and security settings.

What You Can Change

  • Display name and email address
  • Profile avatar / photo
  • Password (current password required)
  • Contact number

Buttons

Save Changes Cancel
  • Save Changes – applies all profile updates; shows a success message
  • Cancel – discards unsaved changes

System Configuration

Admin Only

Control global association settings that appear across the entire system and public website.

Settings Available

  • Association Name – name displayed in the navbar, page titles, and emails
  • Association Logo – upload a new logo (shown in sidebar and public site header)
  • Email Address – official association email for contact forms
  • Phone Number – displayed on the public website
  • Address – office address for the public site
  • Backup Storage Limit (MB) – maximum storage allowed for SQL backup files

Buttons

Save Configuration
  • Save Configuration – saves all settings and refreshes the navbar/logo site-wide
Changing the Association Name updates every page title immediately. Logo changes may require a hard refresh (Ctrl+Shift+R) to appear in the browser cache.

Website Content

Admin Only

Edit the text and media that appear on the public-facing website pages (About Us, Mission, Vision, etc.).

Sections You Can Edit

  • About the Association (history, mission, vision)
  • Hero / banner text and image
  • Contact information displayed on the public site
  • Footer text

Buttons

Save Content Reset
  • Save Content – publishes changes to the live public website
  • Reset – reverts the form to the last saved version
Account

Logout

All Roles

The Logout button is at the bottom of the sidebar. Clicking it ends your session and redirects you to the login page.

For security, always log out when using a shared computer. The session is also automatically ended after a period of inactivity.

Change Password Prompt

If your account has a temporary password, the system will redirect you to a Change Password page immediately after login before you can access any other page. This is a one-time security step.
